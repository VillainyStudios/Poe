Class: Poe.List

Defined in: coffee/List.coffee
Inherits: Poe.Paragraph

Overview

Poe.List is a Poe.Paragraph subclass for bulleted or numbered lists. Instead of a Poe.Line, it takes a Poe.ListItem as a child. However note that a Poe.Line could be added as a child if no bullet or number is wanted.
