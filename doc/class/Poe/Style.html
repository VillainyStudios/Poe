Class: Poe.Style

Defined in: coffee/Style.coffee

Overview

Base class for all styles including TextStyle and paragraphStyle.

Direct Known Subclasses

Poe.ParagraphStyle Poe.TextStyle

Instance Method Summary

  • # (Poe.Style) setTextCursor(textCursor) Set the Poe.TextCursor that this style would use when applying a style to the TextObject that contains the cursor.
  • # (Poe.Style) clone(style) Override in extended class.
  • # (Poe.Style) changed(callbackFn) Register a callback to be called when the style has changed.
  • # (void) hasChanged() Convenience function to call all registered changed callbacks.
