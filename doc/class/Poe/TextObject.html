Class: Poe.TextObject

Defined in: coffee/TextObject.coffee

Overview

Poe.TextObject is the base on which all document children are made of. It serves as a wrapper around a DOM element, in order to get away from the nitty gritty details of the DOM. It is not intended to be instantiated, but instead extended.
